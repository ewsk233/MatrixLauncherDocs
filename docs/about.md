# 关于

## 关于 MatrixLauncher

MatrixLauncher 是一个使用 Kotlin 与 Compose Desktop 开发的 Minecraft 服务器定制启动器框架。

这个项目希望把版本解析、文件校验、客户端更新、账号管理和游戏启动等通用能力整理成可复用模块，让服务器开发者可以将更多精力放在自己的品牌、界面和玩家体验上。

项目目前仍处于持续开发阶段。如果你在使用中发现问题，或对功能设计有建议，欢迎通过下方方式联系。
